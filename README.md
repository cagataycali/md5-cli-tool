# Installing

```
npm i -g md5-cli
```

# Usage

```
md5 encrypt me
# Output: ab86a1e1ef70dff97959067b723c5c24

md5 decrypt ab86a1e1ef70dff97959067b723c5c24
# Output: me
```

# Use self hosted

```
MD5=https://ownapp.herokuapp.com md5 encrypt me
```
